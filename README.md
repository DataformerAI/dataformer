# Dataformer

Solving data for LLMs - Create, Curate & Clean.

## Installation

Github Source:
```
pip install dataformer@git+https://github.com/DataformerAI/dataformer.git 
```

Using Git:
```
git clone https://github.com/DataformerAI/dataformer.git
cd dataformer
pip install .
```
